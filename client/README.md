# Frontend

This folder contains the **frontend (client-side)** code for the project.  

---

## Folder Purpose

The `client/` application is responsible for:
- Rendering the user interface
- Handling user interactions
- Making API requests to the backend
- Managing frontend state and routing

---

## Tech Used

- **Framework / Library:** React / Vite
- **Language:** JavaScript
- **Styling:** CSS / Tailwind
- **Routing:** React Router

---

## Run frontend locally
```
AI-Resume-Builder> cd client
AI-Resume-Builder/client> npm install
AI-Resume-Builder/client> npm run dev
```
